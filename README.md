# Sales Performance Dashboard

Enterprise dashboard with 12+ KPIs, YoY growth, moving averages, and automated refresh

## Features
- 12+ KPIs tracked (Revenue, YoY Growth, Moving Averages, Variance)
- Real-time regional performance analysis
- 15% revenue uplift opportunity identification
- Automated SQL procedures (10+ hours/week saved)
- Interactive visualizations with Plotly
